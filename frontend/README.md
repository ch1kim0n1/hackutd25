# Frontend

This directory contains all the source code for the React-based frontend application.

**Stack:**
- **React:** UI library
- **Electron:** Framework for native desktop application
- **Tailwind CSS:** Utility-first CSS framework

**Structure:**
- `src/`: Main source code
- `src/components/`: Reusable UI components
- `src/pages/`: Top-level page components
- `src/hooks/`: Custom React hooks
- `src/contexts/`: Global state management
- `src/styles/`: Global styles and Tailwind CSS
- `src/assets/`: Static assets (images, fonts)
- `tailwind.config.js`: Tailwind CSS configuration
- `postcss.config.js`: PostCSS configuration
